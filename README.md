## 📈 학습 방식

1. **개념 학습** — 알고리즘 이론 정리/ 노션 정리
2. **유형 파악** — 문제 유형·태그 확인 후 접근 전략 수립
3. **직접 풀이** — 힌트 없이 시간 제한 두고 풀기
4. **복기 & 최적화** — 풀이 후 다른 접근법·시간복잡도 비교
5. **언어 교차 구현** — Python으로 먼저 풀고, C++로 재구현하여 C++ 학습
